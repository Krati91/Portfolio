Crimson Reign

About

Crimson Reign is a hack-and-slash game set in a gritty medieval world, where players engage in fast-paced melee combat while navigating perilous environments. The game features AI-controlled enemies, immersive combat mechanics, and an interactive world that provides a dynamic experience for players.

Project

  • Time Frame:
  • Engine:

Engaging melee combat system with animations and user input. Enemy AI with patrol, chase, and attack behaviors.HUD with health and stamina bars. Interactive environments, including doors, triggers, and platforms

Implemented object-oriented programming concepts like inheritance and polymorphism. Coded custom character classes, including player and enemy logic. Damage system with health and hit points. Game saving and loading functionality.

Used interpolation for floating platforms. Utilized timers and debug drawing for gameplay logic testing.

Built a foundation in Unreal Engine C++ through practical implementation of game systems. Learned to manage complex game logic and ensure seamless gameplay experiences.
Check out the repository at GitHub for more details and source code.
