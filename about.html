Who am I?

With a deep understanding of object-oriented programming, game mechanics, and real-time rendering, I am dedicated to crafting immersive, high-performance gaming experiences. Although I am at the beginning of my professional journey, my academic projects and personal initiatives have allowed me to hone my skills in designing interactive gameplay systems, optimizing performance, and creating visually stunning worlds. I am constantly eager to learn, adapt, and collaborate within dynamic teams to bring innovative ideas to life.

Courses

What I have learnt?

UNREAL ENGINE 5 C++ MULTIPLAYER SHOOTER (Ongoing)

This course teaches multiplayer game development in Unreal Engine, covering matchmaking, network replication, and lag compensation techniques. It includes multiplayer gameplay elements like weapons, pickups, and actions, along with creating a functional HUD.
Learning about core multiplayer classes, custom game modes, and managing online sessions using the Unreal Engine Online Subsystem, as well as displaying dynamic in-game announcements.

UNREAL ENGINE 5 C++ THE ULTIMATE GAME DEVELOPER COURSE

This course covers creating an RPG action game in Unreal Engine 5, with third-person melee combat, enemy creation, and player stats. It includes using UE5's Open World system, Quixel Megascans for realistic scenes, and Motion Warping for custom animations.
The course also covers coding best practices, data structures, MetaSounds for high-quality audio, and designing various enemies. It teaches importing dungeon levels into the Open World using Packed Level Instances.

UNREAL ENGINE 4 BLUEPRINTS - THE ULTIMATE DEVELOPER COURSE

This course allows to create three complete games, including a mobile game, and test them on their devices. It teaches how to find free online assets for game props, characters, and environments. Learnt Unreal Engine's Blueprint scripting system to develop gameplay without needing to learn coding.

UNREAL ENGINE 5 C++ MULTIPLAYER: MAKE AN ONLINE CO-OP GAME

This course teaches intermediate C++ and the latest UE5 technology to create online multiplayer adventure games. Learnt the fundamentals of online multiplayer game development, including creating gameplay sessions and mastering Unreal’s replication system.
The course covers using the Online Subsystem to create online sessions, integrating Steam for online connectivity, and designing a 2-player co-op adventure game. Additionally, learnt level design techniques for their multiplayer games.

UNREAL ENGINE 5 C++ DEVELOPER: LEARN C++ & MAKE VIDEO GAMES

This course teaches how to create games in Unreal Engine 5 using C++, the industry standard programming language. Learnt Object-Oriented Programming, game design principles, and AI behavior programming for enemies.
The course also covers programming patterns, best practices, when to use Blueprint or C++, and the Unreal Gameplay Framework. Gained strong problem-solving skills and knowledge of modern game development technologies and techniques.

THE ULTIMATE GIT COURSE - WITH APPLICATIONS IN UNREAL ENGINE

This course covers Git basics, using Git from the command line and tools like GitHub, Visual Studio, and VS Code. Learnt to track Unreal Engine and C++ projects with Git in UE4 and UE5.
It also teaches advanced Git features like branching, merging, rebasing, reverting versions, and resolving merge conflicts. Gained experience in repository management and customizing Git with preferred text editors.

UNREAL ENGINE 5 BLUEPRINTS: FIRST PERSON SHOOTER (FPS)

This course covers creating a first-person character with movement and animation, as well as weapons and combat systems with sound and particle effects. Students will design hostile AI drones that pursue and attack the player.
It also includes creating a challenging timed game where the goal is to clear the area of drones, implementing Game Mode, HUD with timers, health bars, and score, and adding pickups like health and ammo.

C++ FUNDAMENTALS: GAME PROGRAMMING FOR BEGINNERS

This course teaches how to write games in C++ from scratch, covering Object-Oriented Programming (OOP) basics, programming fundamentals like variables, loops, and if-statements. Learnt clean coding practices, how to compile and run C++ games, and problem-solving and debugging techniques.
The course also covers using the Raylib game library for simple games and how to animate 2D characters.

LEARN C++ FOR GAME DEVELOPMENT

This course helps to start programming in Unreal Engine with C++ by breaking down advanced techniques into manageable steps. Gained a solid understanding of C++ basics and apply their knowledge through a combination of theory and hands-on programming in lab-style sessions.

MATH FOR VIDEO GAMES: THE FASTEST WAY TO GET SMARTER AT MATH

This course covers key math concepts for game development, including number base manipulation, arithmetic in multiple bases, and IEEE754 floating point.

UNREAL ENGINE 4 C++ THE ULTIMATE GAME DEVELOPER COURSE
